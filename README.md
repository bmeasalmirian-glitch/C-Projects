# Windows Forms Application (.NET Framework)

This project is a **desktop application** developed using **C#** and **Windows Forms** based on the provided project specifications.

##  Overview
This application demonstrates the fundamentals of **event-driven programming** and **UI development** in Windows Forms.  
It was designed and implemented as part of a course project.


##  Features
- Interactive graphical interface built with Windows Forms  
- Event-based logic handling (buttons, text inputs, etc.)  
- Structured code in C# with simple and reusable components


##  Technologies Used
- **Programming Language:** C#  
- **IDE:** Visual Studio  
- **GUI Library:** Windows Forms
